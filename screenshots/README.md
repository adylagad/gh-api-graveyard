# Screenshots for Submission

Place the following screenshots in this directory:

1. **01-installation.png** - Terminal showing `gh extension install adylagad/gh-api-graveyard`
2. **02-scan.png** - Running `gh api-graveyard scan` with output
3. **03-report.png** - Generated `api-graveyard-report.md` preview
4. **04-dry-run.png** - `gh api-graveyard prune --dry-run` output
5. **05-prune.png** - `gh api-graveyard prune` execution
6. **06-before-after.png** - Side-by-side OpenAPI spec comparison
7. **07-pull-request.png** - GitHub PR created by the tool

## Tips for Great Screenshots

- Use a clean terminal with good contrast
- Include relevant context (command + output)
- For the report, show the markdown rendered view
- For PR, show the description with confidence scores
- Capture full terminal output when possible
